# Weather Dashboard

## About

This application presents the weather from a list of cities via the Openweather API

## Mockup

![Weather Dashboard Mockup](./assets/06-server-side-apis-homework-demo.png)


## Features

The user clicks on the get weather button. If the city name is in the searchistory, bring it to the top.
Users can choose among three systems of measurement:
"metric" > Metric (°C)
"imperial" > Imperial (°F)
"standard" > Standard (°K)
